# Jogo Super Mario

Este é um jogo simples inspirado no Super Mario, onde o jogador controla um personagem que pula sobre obstáculos.

## Como Jogar

1. **Controles**: Pressione a barra de espaço para pular.
2. **Objetivo**: Evite os obstáculos e colete pontos.
3. **Dificuldade**: A velocidade dos obstáculos aumenta a cada 5 pontos.

## Funcionalidades

- **Pulo**: O personagem pode pular para evitar obstáculos.
- **Pontuação**: A pontuação aumenta à medida que você evita obstáculos.
- **Dificuldade Aumentada**: A velocidade dos obstáculos aumenta a cada 5 pontos.
- **Reiniciar**: Após o jogo acabar, você pode reiniciar clicando no botão "Reiniciar".

## Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em um navegador web.

Divirta-se jogando!
